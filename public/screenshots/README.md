# PWA Screenshots

This folder contains screenshots for the Progressive Web App (PWA) install prompt.

## Required Screenshots

### Desktop/Wide Screenshot
- **Filename**: `desktop.png`
- **Size**: 1280x720 pixels (or similar 16:9 ratio)
- **Purpose**: Shows how the app looks on desktop/tablet

### Mobile/Narrow Screenshot
- **Filename**: `mobile.png`
- **Size**: 750x1334 pixels (iPhone size, or similar portrait ratio)
- **Purpose**: Shows how the app looks on mobile devices

## How to Create Screenshots

1. **Desktop Screenshot**:
   - Open your portfolio in a browser at full width
   - Take a screenshot of the home page
   - Crop to 1280x720 or similar wide format
   - Save as `desktop.png`

2. **Mobile Screenshot**:
   - Open your portfolio in mobile view (Chrome DevTools)
   - Set viewport to iPhone (750x1334) or similar
   - Take a screenshot of the home page
   - Save as `mobile.png`

## Tips

- Show your best content (home page or portfolio section)
- Ensure text is readable
- Remove browser UI elements (address bar, etc.)
- Use high-quality images
- These screenshots appear in app stores and install prompts

## Note

Screenshots are optional but recommended for a better PWA install experience. They will appear in:
- Chrome/Edge install prompts
- Android app store listings (if you publish to Play Store)
- iOS install banners
